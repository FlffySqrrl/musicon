# MusicOn

![MusicOn Logo](https://github.com/chriscyang/musicon/blob/master/musicon/static/img/musicon.png)

## Requirements

* `python 2.7`
* `django 1.5`
* `mysql`

## Setup

```
$ python manage.py collectstatic
$ python manage.py syncdb
$ python manage.py validate
$ python manage.py runserver
```
